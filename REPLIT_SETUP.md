# Quick Replit Setup (2 minutes)

## Steps:

1. **Go to Replit.com** and sign up (free)

2. **Import from GitHub:**
   - Click "Create Repl"
   - Select "Import from GitHub"
   - Paste: `https://github.com/jayeshsv11/TSFAM-Silime.git`
   - Click "Import from GitHub"

3. **Run instantly:**
   - Replit will auto-detect Python
   - Click the big green "Run" button
   - Your script runs in the cloud!

4. **Keep it running 24/7:**
   - Click "Deploy" button (top right)
   - Choose "Reserved VM" or "Autoscale"
   - Free tier available

**That's it!** Your device is free, scripts run on Replit's servers.

## Alternative - If Selenium needed:
Replit might not support browser automation. Use **PythonAnywhere** instead:
1. Go to pythonanywhere.com
2. Sign up free
3. Upload files via "Files" tab
4. Set up scheduled task
